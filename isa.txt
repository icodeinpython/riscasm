- Word size: 32 bit registers, 32 bit data bus, 32 bit instructions
- Byte ordering: little endian
- Register file: 32 bit general purpose registers X0-X31
  - X0 is just 0, any writes are no-ops
- 32 bit program counter

Instruction Set:
- R-Type - register: used for aritmetic and logical operations involving 3 registers
  - Opcode - instruction type: 6 bits
  - Rs - source register 1: 5 bits
  - Rt - source register 2: 5 bits
  - Rd - dest register: 5 bits
  - Funct - specific function (add, sub, and, xor): 11 bits
- I-type - intermediate: used for constants, or memory loads/stores
 			 where the immediate is an offset
  - Opcode - instruction type: 6 bits
  - Rs - Source register: 5 bits
  - Rd - target register: 5 bits
  - Immediate - sign extenended constant
- J-type - jump: used for various kinds of unconditional jumps
  - Opcode - jump instruction: 6 bits
  - Address - address to jump to: 26 bits (lower 26 bits, if more bits
				  are needed, use a register with 0 offset)

All instructions:
- R-Type:
  - ADD Rd, Rs, Rt; Rd = Rs + Rt
  - Sub Rd, Rs, Rt; Rd = Rs - Rt
  - AND Rd, Rs, Rt; Rd = Rs & Rt
  - OR Rd, Rs, Rt; Rd = Rs | Rt
  - XOR Rd, Rs, Rt; Rd = Rs ^ Rt
  - SLL Rd, Rs, Rt; Rd = Rs << Rt
  - SRL Rd, Rs, Rt; Rd = Rs >> Rt
  - SRA Rd, Rs, Rt; Rd = Rs >> Rt (sign extended)
  - SLT Rd, Rs, Rt; Rd = (Rs < Rt) ? 1 : 0

- I-Type:
  - ADDI Rt, Rs, Imm; Rt = Rs + SignExtend(Imm)
  - ANDI Rt, Rs, Imm; Rt = Rs & ZeroExtend(Imm)
  - ORI Rt, Rs, Imm; Rt = Rs | ZeroExtend(Imm)
  - LUI Rt, Imm; Rt = Imm

- I-Type memory:
  - LW Rt, Offset(Rs)
  - SW Rt, Offset(Rs)

- I-Type control flow:
  - BEQ Rs, Rt, Offset; if Rs==Rt, PC = PC + 4 + SignExtend(Ofser)
  - BNE Rs, Rt, Offset; if Rs!=Rt, PC = PC + 4 + SignExtend(Ofser)

- J-type:
  - J Target; PC = PC[high] | Target
  - JAL Target; X31 = PC + 4, PC = PC[high] | Target
  - JR Rs; PC=Rs


Reg Codes:
  x0 -> 0
  x1 -> 1
  x2 -> 2
  ...
  x31 -> 31 (0b11111)
